Инструкция

1. Смещение
    Чтобы переместить фигуру, укажите смещение dx по оси Ox,
    смещение dy по оси Oy и нажмите кнопку 'Выполнить смещение'.

2. Масштабирование
    Чтобы отмасштабировать фигуру, укажите координаты центра 
    масштабирования, коэффициент масштабирования и нажмите кнопку
    'Выполнить масштабирование'.

3. Поворот
    Чтобы отмасштабировать фигуру, укажите координаты центра 
    поворота, угол в градусах и нажмите кнопку
    'Выполнить масштабирование'.

4. Отменить последнее преобразование
    Можно отметить все преобразования над фигурой вплоть до исходного
    состояния.

5. Вернуть исходное состояние.
    Возвращает фигуру в изначальное состояние.

6. Экспорт / импорт точек

    a. Сохранение базовых точек спутника:

        Файл -> Сохранить данные -> Выберите нужный файл.

    b. Загрузка базовых точек спутника:

        Файл -> Импортировать данные -> Выберите нужный файл.

        Для построения спутника необходимо 12 точек:

        точка 1: Центр иллюминатора
        точка 2: Верхняя точка иллюминатора 
        точка 3: Левая крайняя точка иллюминатора 
        точка 4: Верхняя точка передней стенки спутника (окружности)
        точка 5: Левая крайняя передней стенки спутника
        точка 6: Крайняя точка верхней антенны
        точка 7: Нижняя правая точка верхней солнечной батареи
        точка 8: Нижняя левая точка верхней солнечной батареи
        точка 9: Верхняя правая точка верхней солнечной батареи
        точка 10: Центр задней стенки спутника (полуэллипса)
        точка 11: Верхняя точка задней стенки спутника
        точка 12: Левая крайняя точка задней стенки спутника

ПРИМЕЧАНИЕ:
* В программе под центром спутника подразумевается точка, которая находится
по середине между центром передней стенки спутника (окружности) и крайней
левой точкой задней стенки (полуэллипса). Данная точка выбрана из эстетических
соображений. Не путать с центром масс и т.д.
