Инструкция

1. Добавление точек
    a. С помощью кнопки 'Добавить точку'
        Введите координаты точки в соответствующие поля x и y,
        укажите, к какому множеству данная точка относится, и
        нажмите кнопку 

    b. С помощью кнопок мыши
        При нажатии на координатную плоскость левой кнопкой 
        мыши добавляется точка в множество точек А 
        (помечаются красным цветом), при нажатии правой 
        кнопкой - во второе множество B (помечаются синим)

    c. Чтение из файла
        Файл -> Импортировать данные -> Выберите нужный файл

2. Удаление точек
    a. Из таблицы
        Выберите точку в таблице правой кнопкой мыши и 
        подтвердите удаление.

3. Перемещение по плоскости
    Чтобы перемещаться по плоскости, зажмите среднюю кнопку мыши.

4. Сохранение точек
    Файл -> Экспортировать данные -> Выберите нужный файл.

5. Выполнение программы
    Чтобы программа смогла найти корректное ненулевое
    решение, необходимо, чтобы в каждом множестве было минимум 3 точки.
