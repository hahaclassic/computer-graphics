Инструкция

1. Добавление точек
    a. С помощью кнопки 'Добавить точку'
        Введите координаты точки в соответствующие поля x и y,
        укажите, к какому множеству данная точка относится, и
        нажмите кнопку 
    b. С помощью кнопок мыши
        При нажатии на координатную плоскость левой кнопкой мыши
        добавляется точка в первое множество точек (помечаются 
        синим цветом), при нажатии правой кнопкой - во второе 
        множество (помечаются красным)
    c. Чтение из файла
        Файл -> Импортировать данные -> Выберите нужный файл

2. Удаление точек
    Чтобы удалить точку, нажмите на нее средней кнопкой мыши

3. Сохранение точек
    Файл -> Экспортировать данные -> Выберите нужный файл
    
4. Выполнение программы
    Чтобы программа смогла найти корректное ненулевое решение, 
    необходимо, чтобы в каждом множестве было минимум 3 точки.
